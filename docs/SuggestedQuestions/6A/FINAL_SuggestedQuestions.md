# FINAL: Suggested Questions (追问推荐) 项目总结报告

## 项目成果
成功为 RAG 系统引入了“智能追问推荐”能力：
1. **交互深度提升**：用户在获得答案的同时，系统会自动联想并提供 3 个高相关度问题，极大降低了用户二次交互的思考成本。
2. **生成架构稳健**：采用延迟加载与容错解析设计，确保生成追问不会阻塞主检索链路，且对 LLM 输出抖动具备极高的容错性。
3. **协议扩展性**：通过对 `MCPToolResponse` 的元数据扩展，为后续前端 UI 渲染更丰富的组件（如点击气泡）打下了基础。

## 技术细节
- **解析引擎**：结合正则提取与 JSON 解析，能够处理常见的 LLM 格式溢出。
- **逻辑剥离**：生成逻辑封装于 `SuggestedQuestionGenerator`，易于在不同工具间复用或进行独立的单元测试。

## 操作指引
用户提问后，可在回答末尾看到“💡 您可能还想问：”版块。此版块可通过 `settings.yaml` 中的 `enable_suggested_questions: false` 全局关闭。
