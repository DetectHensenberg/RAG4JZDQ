# ALIGNMENT — 需求4：标书智能化审查助手

## 已确认决策

| # | 问题 | 决策 |
|---|------|------|
| 1 | 交互方式 | **分步向导页 (Step Wizard)**：Step1 上传招标文件 → Step2 确认废标项清单(可编辑) → Step3 上传投标文件 → Step4 审查报告 |
| 2 | 废标项识别 | **规则 + LLM 混合**：先用正则/关键词粗筛（★、否则废标、必须、资格性审查等），再用 LLM 精化分类和提取具体要求 |
| 3 | 审查报告 | **流式 Markdown + 结构化表格**：SSE 流式输出报告，完成后解析为表格，双 Tab 查看 |
| 4 | 清单编辑 | **可编辑**：LLM 识别后用户可增删改废标项条目，确认后进入核查 |

## 不变约束

- 不影响需求1、需求2功能
- 复用暗黑毛玻璃 UI 风格
- 作为 BidAssistant 下拉菜单的一个选项
- Git 提交由用户手动控制
