# 需求3：标书商务文件编写助手

> **优先级：低**

## 一、背景

707部每年都会都大量的标书编写工作，但大部分的标书商务文件都是类似的要求，需求不断重复的复制/粘贴相关内容，如：
- 企业资质证书
- 财务报表
- 投标相关声明函
- 违法失信证明（官方网站截图）
- 售后服务等

比较浪费人力和时间。

## 二、具体需求

通过智能体应用平台读取各类招标文件要求，并识别出商务文件要求的条款，并根据公司既定投标文件模板生成商务文件大纲，并调用投标知识库，对部分重复性内容进行填充，辅助标书编写人员快速编制商务文件。

同时能够实现检索到的类似于身份证、营业执照、资质证书等文件能自动打上水印"仅限于XXX（项目编号）项目投标使用"，规范通用资质及材料的使用和管理。

## 三、场景举例

1. 707部获取到了A客户的智慧城市集成项目的招标文件，要求商务文件部分需要编写公司介绍、公司规模、提供法人代表的身份证扫描件、营业执照、资质证书、相关承诺和声明等。

2. 标书编写人员将该招标文件上传至智能体应用平台后，平台能自动识别并读取到商务文件的条款要求，并根据既定投标文件模板格式生成商务文件大纲。

3. 平台调用投标知识库中的法人代表身份证扫描件、营业执照等信息，打上水印后，填充到投标文件对应章节中。

4. 若标书中有相关承诺书模版，平台会根据招标文件模版自动填充内容。

## 四、核心功能点

| 功能 | 描述 |
|------|------|
| 招标文件解析 | 自动读取并识别商务文件条款要求 |
| 大纲生成 | 根据既定模板生成商务文件大纲 |
| 知识库调用 | 自动填充重复性内容 |
| 水印添加 | 为证照文件自动添加项目专用水印 |
| 模板填充 | 根据招标文件模板自动填充承诺书等内容 |

## 五、预期效果

- 减少重复性复制粘贴工作
- 提高标书编写效率
- 规范资质材料的使用和管理
- 降低人工错误率
