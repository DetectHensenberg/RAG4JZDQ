# ACCEPTANCE — 需求3：标书商务文件编写助手

## 验收标准

### AC-1: 模块入口
- [ ] 标书助手下拉菜单显示"需求3：标书商务文件编写助手"
- [ ] 点击后进入商务文件编写页面，不再显示"开发中"

### AC-2: 招标文件上传
- [ ] 支持拖拽上传 PDF/DOCX 文件
- [ ] 上传成功后显示文件名和大小
- [ ] 创建编写会话，返回 session_id

### AC-3: 条款提取
- [ ] 点击"开始解析"后，LLM 流式提取商务文件条款
- [ ] 条款以列表形式展示（序号、标题、描述、分类、是否必须）
- [ ] 用户可勾选/取消勾选条款

### AC-4: 大纲生成
- [ ] 点击"生成大纲"后，根据选中条款生成结构化大纲
- [ ] 大纲支持拖拽排序
- [ ] 大纲显示知识库匹配状态（✓ 已匹配 / ✗ 待补充）

### AC-5: 内容填充
- [ ] 点击"开始填充"后，流式填充各章节内容
- [ ] 实时显示填充进度和内容预览
- [ ] 支持手动编辑填充后的内容

### AC-6: 水印添加
- [ ] 输入项目编号后，可选择需要添加水印的材料
- [ ] 点击"添加水印"后，后端处理并返回带水印的文件
- [ ] 水印文字格式："仅限于 {项目编号} 项目投标使用"

### AC-7: 导出功能
- [ ] 点击"导出"后，生成 Word 文档
- [ ] 文档包含完整大纲结构和填充内容
- [ ] 证照附件以图片形式嵌入文档

### AC-8: 投标材料管理
- [ ] 可上传投标材料（资质证书、财务报表、声明函、证照等）
- [ ] 材料列表支持分类筛选
- [ ] 材料支持设置有效期

### AC-9: 模板管理
- [ ] 可创建/编辑承诺书、声明函等模板
- [ ] 模板支持变量占位符（如 {{公司名称}}、{{项目名称}}）

### AC-10: 会话管理
- [ ] 可查看历史编写会话
- [ ] 可继续未完成的会话
- [ ] 可删除会话

## 测试用例

### T-1: 完整流程测试
1. 上传招标文件（测试用 PDF）
2. 确认提取的条款
3. 生成并调整大纲
4. 填充内容
5. 添加水印
6. 导出 Word 文档
7. 验证文档内容完整性

### T-2: 水印测试
1. 上传图片格式证照（JPG/PNG）
2. 添加水印
3. 验证水印文字正确、位置合理

### T-3: PDF 水印测试
1. 上传 PDF 格式证照
2. 添加水印
3. 验证每页都有水印

### T-4: 材料管理测试
1. 上传多个材料
2. 按分类筛选
3. 删除材料
4. 验证 ChromaDB 同步

### T-5: 模板测试
1. 创建承诺书模板（含变量）
2. 在填充时使用模板
3. 验证变量替换正确
